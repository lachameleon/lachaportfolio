# lacha.dev

A static personal website featuring a developer portfolio, project links, and a friends section with pixelated badges.

## Structure
- `index.html` - Main page with bio, projects, and friends
- `mail.html` - Contact/mail page
- `files/base.css` - Base dark theme styling
- `files/home.css` - Badge and layout styling
- `files/js/oneko.js` - Cat cursor follower
- `files/js/ko-fi.js` - Donation widget
- `files/img/buttons/` - Friend badge images

## Development

This is a static site with no build process. Files are served directly.

To view locally:
1. Use Five Server VS Code extension for live reload
2. Modify HTML/CSS/JS directly
3. Changes are reflected immediately
